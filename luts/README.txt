Drop .cube 3D LUT files here for professional color grading.

The tool automatically scans this folder and adds any .cube files
to the LUT preset dropdown in the Visual Effects tab.

File format: Standard .cube (LUT_3D_SIZE header)
Recommended size: 33x33x33 or 65x65x65

You can download free LUTs from:
- https://www.lutify.me/free-luts/
- https://www.freeluts.net/
